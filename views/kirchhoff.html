Fórmulas de Millman — tensión de nodo
V_nodo = (V1/R1 + V2/R2 + V3/R3 + ...) / (1/R1 + 1/R2 + 1/R3 + ...) Ejemplo: 3 ramas V1=12V/R1=100Ω, V2=5V/R2=200Ω, V3=0/R3=50Ω Numerador: 12/100 + 5/200 + 0/50 = 0.12+0.025+0 = 0.145 Denominador: 1/100 + 1/200 + 1/50 = 0.01+0.005+0.02 = 0.035 V_nodo = 0.145/0.035 = 4.14V
