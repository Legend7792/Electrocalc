⏱ Timer 555

Modo astable (oscilador) y monoestable (disparo único).

f = 1.44 / ((Ra + 2×Rb) × C) T_high = 0.693 × (Ra + Rb) × C T_low = 0.693 × Rb × C Duty = (Ra + Rb) / (Ra + 2×Rb) × 100%
El 555 astable no puede generar duty cycle < 50% con el circuito clásico. Para <50% se necesita un diodo.
