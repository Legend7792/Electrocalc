⚡ Ley de Ohm y Potencia
V = I × R | I = V/R | R = V/I P = V × I = I²×R = V²/R En AC: P = V×I×cos(φ) | S = V×I | Q = V×I×sin(φ) | S² = P² + Q²
🔶 Resistencias
Serie: Rt = R1+R2+...+Rn Paralelo: 1/Rt = 1/R1+1/R2+... (2 res: Rt = R1×R2/(R1+R2)) Divisor V: Vout = Vin × R2/(R1+R2) Divisor I: I1 = It × R2/(R1+R2)
⊣⊢ Condensadores
Serie: 1/Ct = 1/C1+1/C2+... Paralelo: Ct = C1+C2+...+Cn Carga: Vc(t) = Vcc×(1−e^(−t/τ)) τ = R×C Energía: E = ½×C×V² Reactancia: Xc = 1/(2π×f×C)
〰️ Inductores
Serie: Lt = L1+L2+... Paralelo: 1/Lt = 1/L1+1/L2+... RL: iL(t) = (V/R)×(1−e^(−t/τ)) τ = L/R Energía: E = ½×L×I² Reactancia: XL = 2π×f×L
📡 Resonancia y Filtros
fr = 1/(2π√(LC)) ω0 = 2π×fr Q = ω0×L/R = 1/(ω0×C×R) Filtro RC Paso Bajo: fc = 1/(2π×RC) H = 1/√(1+(f/fc)²) Filtro RC Paso Alto: fc = 1/(2π×RC) H = (f/fc)/√(1+(f/fc)²) Atenuación: dB = 20×log10(H)
⏱ Timer 555
Astable: f = 1.44/((Ra+2×Rb)×C) Duty = (Ra+Rb)/(Ra+2×Rb) Monoestable: t = 1.1×R×C
📐 Op-Amp
Inversor: Av = −Rf/Rin Vout = −(Rf/Rin)×Vin No inversor: Av = 1+Rf/R1 Vout = (1+Rf/R1)×Vin Sumador inv: Vout = −Rf×(V1/R1+V2/R2+...) Diferencial: Vout = (Rf/R1)×(V2−V1) dB = 20×log10(|Av|)
🔺 Transistor BJT
IC = β×IB IE = IC+IB β = hFE = IC/IB VCEsat ≈ 0.2V VBE ≈ 0.7V (Si) Divisor tensión: VB = Vcc×R2/(R1+R2) VE = VB−VBE IC = VE/RE
🔄 Transformador (ideal)
n = N1/N2 = V1/V2 = I2/I1 P1 = P2 (sin pérdidas) η = Pout/Pin × 100%

Tensiones de seguridad (SELV)
SELV (Safety Extra-Low Voltage): AC: ≤ 50V rms DC: ≤ 120V Considerar peligroso por encima de estos límites. EPI mínimo recomendado: guantes dieléctricos, botas aislantes, gafas. Regla de oro: NUNCA trabajar en tensión. Verificar ausencia de tensión SIEMPRE.
